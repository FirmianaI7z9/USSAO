概要

地理オリンピックとは？

地理オリンピックとは、世界の地誌や系統地理などの地理に関する知識を問う科学オリンピック競技です。 比較的新しい大会です。

国内大会への参加方法・国内大会の流れ

地理オリンピックの国内大会は「科学地理オリンピック日本選手権(JGeO)」です。公式の開催情報は以下リンクから確認できます。

科学地理オリンピック日本選手権 公式

https://japan-igeo.com/

参加申し込み 手段1 : 科学オリンピック共通サイト(下記)から個人申し込み
手段2 : 学校での一括申し込み

科学オリンピック共通サイト

https://contest-kyotsu.com/entry/

参加費 第1次選抜 : 無料
第2次選抜 : 交通費等の情報求む
第3次選抜 : 交通費等の情報求む
年齢制限 第1次選抜開催年の4月以降に大学教育を受けていない19歳未満
出題範囲 調査中
問題の一部は英語で出題される
大会形式 第1次選抜 : オンラインでのマルチメディアテスト(選択式)(上位100人のみ二次選考進出)
第2次選抜 : 対面での記述式試験(高2以下で上位約10人程度が第3次選抜に進む)
第3次選抜 : フィールドワーク等
受賞 第2次選抜の成績で表彰。高3受賞可能。

科学地理オリンピック日本選手権への申し込みから大会参加までの流れは次のようになっています。

1申し込み申し込みサイトか学校一括での申し込み
2受験票受け取り郵送で届く(受験に必要)
3第1次選抜参加自宅等からオンラインで参加
4第1次選抜通過者発表公式サイト上で発表
5第2次選抜参加会場で参加
6第2次選抜成績発表?詳細不明
7第3次選抜参加詳細不明

国際大会への参加方法・国際大会の流れ

地理オリンピックの国際大会は、「国際地理オリンピック(iGeo)」です。

・国際大会の日本代表になるために満たすべき条件 :

国内大会で満たすべき条件 高2以下の第2次選抜参加者のうち、上位約10人になる
代表選抜で満たすべき条件 第3次選抜で上位4人になる
年齢制限 第2次選抜開催年の6月時点で16歳以上19歳以下であること

代表選考は第3次選抜として開催されます。詳細は調査中です。
